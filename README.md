# ⛅ Weather App
An application that shows detailed, real-time weather information for a searched city. The backend was built using **Django** to access the **OpenWeather API** and send the data to the frontend, which was developed with **React**. The project was inspired by the [GeeksforGeeks Weather App with Django tutorial](https://www.geeksforgeeks.org/python/weather-app-using-django-python/) and the video [React Django Tutorial - Learn React Python Django In 1 Hour | For Beginners](https://www.youtube.com/watch?v=xldTxXtNiuk).

# 🛠 Technologies Used
- Backend: Django (Python)
- Frontend: React (JavaScript)
- API: OpenWeather API
- Communication: REST API (JSON)
